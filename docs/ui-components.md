# Компонентная архитектура UI

## I. Ключевые экраны приложения и компоненты UI десктопного приложения

### 1. Главный экран:

- Поле поиска (Search Field) - служит для поиска необходимого чата.
- Кнопка "Создать чат" - кнопка для создания нового чата.
- Иконка аватарки пользователя - открывает всплывающее окно для "Настройки аккаунта" (3).
- Кнопка удаления выбранного чата - удаляет выбранный чат.
- ChatList - список чатов (ChatItem).
- ChatItem --- элемент списка чатов:
  - Название чата.
  - Иконка "..." → открывает всплывающее окно "Редактирование чата" (2).
  - Клик на ChatItem → переход в ChatView.
- Область для введения медицинского запроса:
  - Кнопка "Добавить файл" - кнопка для добавление в запрос файла с рентгеновским снимком.
  - Поле ввода (Input Field) - поле для ввода текстового запроса.
  - Кнопка "Генерация медицинского заключения" - кнопка отправляющая рентгеновский снимок и текстовый запрос в LLM- модель.

### 2. Всплывающее окно "Редактирование чата":

- Кнопка "Переименовать" - кнопка для изменения названия чата.
- Кнопка "Закрепить" - кнопка для закрепления чата выше всех в списке.
- Кнопка "Удалить" - кнопка для удаления чата и всего содержимого.

### 3. Всплывающее окно для "Настройки аккаунта":

- Имя аккаунта с аватаркой.
- Кнопка "Настройки" - кнопка открывает окно для настройки приложения (4).
- Кнопка "Свяжись с нами" - кнопка открывает окно для связи между пользователем и разработчиком (5).
- Кнопка "Выйти" - кнопка закрывает приложение и выходит из аккаунта.

### 4. Окно "Настройки":

- Боковая панель (Sidebar) - содержит названия всех доступных настроек приложения.
- Dropdown "Внешний вид" - при нажатии элемент предоставляет выбор варианта из списка с доступными темами интерфейса.
- Dropdown "Язык интерфейса" - при нажатии элемент предоставляет выбор варианта из списка с доступными языками, на которые переведён интерфейс.
- Кнопка "Удалить чаты" - кнопка удаляет все чаты и их содержимое.
- Кнопка "Выйти" - кнопка закрывает приложение и выходит из аккаунта.

### 5. Окно "Обратная связь":

- Поле ввода "Имя" - поле куда пользователь должен ввести своё имя.
- Поле ввода "Почта" - поле куда пользователь должен ввести имя своей почты, на которую он хочет получить ответ.
- Поле ввода "Сообщение" - поле, в котором пользователь должен максимально подробно должен описать проблем при использовании приложения.
- Кнопка "Отправить" - кнопка отправляющая письмо разработчикам.

## Состояния интерфейса:

### 1. Загрузка.

Во время генерации медицинского заключения на экране появляется иконка загрузки.

![Load_interface](/design/Load_interface.png)

### 2. Ошибка.

При попытке отправки сообщения с незаполненными обязательными полями появляется сообщение об ошибке с красным текстом, указывающее на поле, которое необходимо заполнить.

![Error_interface](/design/Error_interface.png)

### 3. Успех.

При нажатии в окне "Настройки" кнопки "Удалить чаты" на экране появляется уведомление о том что действие прошло успешно.

![Success_interface](/design/Success_interface.png)
